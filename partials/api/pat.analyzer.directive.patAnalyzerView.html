patAnalyzerView
directive in module pat.analyzer

Description

This directive handles the implementation details of embedding an analyzer view in an application. Standard iframes are used as the supporting technology, so there aren't any particular issues to taken into account when using the directive.

Usage

as element:
<pat-analyzer-view
       config="{object}"
       [on-load="{function}"]>
</pat-analyzer-view>
as attribute
<ANY pat-analyzer-view
     config="{object}"
     [on-load="{function}"]>
   ...
</ANY>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
configobject

An analyzer view configuration object.

on-load
(optional)
function

A function to call when the analyzer view loads.
