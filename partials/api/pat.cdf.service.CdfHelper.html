CdfHelper
service in module pat.cdf

Description

The CdfHelper service acts as an abstraction layer so that controllers and directives don't have to directly interface with CDF using $http.

Methods

  • getNewDashboard(path, element)

    This methods fetches a dashboard constructor from the server, from a specified path, and instantiates. It returns a promise that, when resolved, passes the instantiated dashboard to success handler.

    The services makes use of requireJS to assynchronously load the dashboard constructor, and assumes the 'require' global is present in the window object.

    Additionally, most of the times the constructor dependencies rely on certain requireJS configurations, that should be defined in the application. See this link for more information on the requirements to embed pentaho cdf dashboards: CDF Embedding

    Parameters
    ParamTypeDetails
    pathstring

    A path to a dashboard constructor module to be loaded using RequireJS.

    elementelement

    A DOM element or jquery wrapper element where the dashboard should be mounted.

    Returns
    promise

    A promise that will be resolved when the dashboard definition modules is returned and a new dashboard instance is created.
