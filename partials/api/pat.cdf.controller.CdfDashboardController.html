CdfDashboardController
controller in module pat.cdf

Description

This controller is mainly meant to be used by directives that want to load a cdf dashboard and interface with it. It makes use of CdfHelper but it also keeps a register of the controlled dashboard.

Methods

  • getDashboard()

    Gets the controlled dashboard from the controller registry.

    Returns
    dashboard

    The registered cdf dashboard.

  • render()

    Renders the dashboard registered in the controller. Makes use of CdfHelper#renderDashboard.

    Returns
    promise

    A promise that will be resolved when the dashboard finishes rendering.

  • setDashboard(dash)

    Sets the controlled cdf dashboard in the controller registry.

    Parameters
    ParamTypeDetails
    dashdashboard

    A cdf dashboard.

    Returns
    dashboard

    The registered cdf dashboard (for chaining inside promise handlers).

  • setNewDashboard(path, element)

    Gets a new dashboard instance and sets in in the controller registry.

    Parameters
    ParamTypeDetails
    pathstring

    A path with the dashboard definition module.

    elementelement

    A DOM or jquery element where the dashboard should me mounted.

    Returns
    promise

    A promise that will be resolved when the dashboard definition is fetched and a new dashboard is instantianted. The dashboard instance is passed to the promise success handler.
