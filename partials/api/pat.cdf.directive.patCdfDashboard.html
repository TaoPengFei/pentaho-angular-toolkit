patCdfDashboard
directive in module pat.cdf

Usage

as element:
<pat-cdf-dashboard
       path="{string}"
       [parameters="{object}"]
       [events="{object}"]
       [deep-watch="{boolean}"]>
</pat-cdf-dashboard>
as attribute
<ANY pat-cdf-dashboard
     path="{string}"
     [parameters="{object}"]
     [events="{object}"]
     [deep-watch="{boolean}"]>
   ...
</ANY>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
pathstring

A url from where the dashboard definition should be fetched.

parameters
(optional)
object

A map or expressions to bind to dashboard parameters.

events
(optional)
object

A map of expressions to run when a dashboard event is fired. These are usually callback functions.

deep-watch
(optional)
boolean

An expression that, when evaluated, controls whether the parameter expressions should watch deeply or shallowly.
